專為GCP打造的Windows 10 DD鏡像

Written by with ♥ on August 27, 2019 in

前陣子為了要在GCP使用VPS，於是在一段時間裡製作了 Windows DD 鏡像
分別是：
1，中文繁體版 Windows 10 工作站專業版
2，英文版 Windows 10 工作站專業版(尚未上傳)

鏡像具備以下特點

  • 集成 Virtio 驅動以及 Google Virtio Ethernet Adapter 的網卡驅動，因此適用於GCP的VPS伺服器

  • 無需 VNC 交互直接無人值守安裝，DD 完成即可遠程登錄桌面

  • 基於 VOL 版製作，因此可用 KMS 方式激活系統

  • 開啟 Windows 遠程桌面

    中文繁體版
    載點1: http://static.yiy.tw/mirror/dd/win10.gz
    載點2: https://drive.yiy.tw/Mirror/DD/win10.gz

    遠端登入資訊
    帳號: Administrator
    密碼: steveyigame.com
